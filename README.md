**Upload any image → Get instant CIFAR-10 predictions with confidence scores!** Built with Flask, TensorFlow, and your custom-trained AlexNet (76.7% test accuracy).

## Features

- **Image Upload & Preview** - Drag & drop or click to upload
- **Real-time Predictions** - Top predictions with confidence %
- **Beautiful UI** - Responsive design works on all devices  
- **76.7% Accuracy** - Trained on CIFAR-10 dataset
- **10 Classes** - airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck
